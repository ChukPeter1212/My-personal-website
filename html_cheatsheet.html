HTML Cheat Sheet

<!doctype>
Defines HTML5 document type: <!doctype html>
<meta charset>
Specify encoding: <meta charset="utf-8">
<link>
Include CSS or favicons: <link rel="stylesheet" href="styles.css">
<nav>
Semantic container for navigation links.
<header> / <footer>
Semantic page regions.
<main>
Unique main content; only one per page.
<section> / <article>
Group related content. Use <article> for self-contained pieces.
<h1> - <h6>
Headings: use one <h1> per page for accessibility.
<p>
Paragraphs. Avoid using <br> for layout.
<a>
Links. Add rel="noopener" for external links and target="_blank" to open in new tab.
<img>
Provide alt attribute for accessibility. Example: <img src="photo.jpg" alt="Photo description">
<ul> / <ol> / <li>
Lists — use for structured items.
<table>
Tabular data: use <thead><tbody><caption> for accessibility.
<form> <input> <label>
Forms: pair inputs with labels using for and id.
<button>
Interactive button; avoid using <div> as a button.
ARIA roles
Use role attributes to improve screen reader experience when semantic HTML is not enough.
<picture> <source>
Responsive images with different sizes/formats.
<script>
Include JavaScript. Prefer placing before </body> or use defer.

Quick code example

<button id="likeBtn">Like</button>
<script>
  document.getElementById('likeBtn').addEventListener('click', () => {
    alert('Thanks for the like!');
  });
</script>
